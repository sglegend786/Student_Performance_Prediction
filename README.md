# 🎓 Student Performance Prediction

A Machine Learning web application that predicts a student's final performance based on academic factors such as attendance, assignments, quizzes, internal marks, and previous academic performance.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/sglegend786/Student_Performance_Prediction.git
```

### Navigate to the Project Folder

```bash
cd Student_Performance_Prediction
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
